# Beedoo QA Challenge

Este repositório contém a resolução do desafio para a vaga de Analista de Qualidade de Software.

## Estrutura do projeto

docs → análise da aplicação  
bugs → relatório de bugs  
test-cases → casos de teste  
evidencias → prints da execução  

## Casos de teste

Os cenários de teste foram documentados na planilha:

[https://docs.google.com/spreadsheets/d/1Ovqc9Uf2BG8F09N_-ChufLzjq4c3eDinfUzsW-yA6qc/edit?usp=sharing]

## Estratégia de testes

A estratégia adotada foi baseada em testes funcionais manuais focados em:

- validação de campos
- cenários positivos
- cenários negativos
- testes de limite

As evidências da execução dos testes estão disponíveis na pasta evidencias.
